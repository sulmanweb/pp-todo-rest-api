## To Run Locally

- clone the app
- make sure you have ruby 2.6.3
- run `bundle`
- run `rails db:create` (make sure postgresql installed and set `config/database.yml` according to your system)
- run `rails db:migrate`
- run `bin/rspec` to test the app
